# 🚀 Cosmic Dodger

🎮 **Live Demo:** https://alexisontheway.github.io/cosmic-dodger/

Cosmic Dodger is a fast-paced browser-based arcade game where players control a spacecraft and dodge incoming obstacles for as long as possible. The game focuses on smooth gameplay, responsive controls, and real-time collision mechanics.

---

## 🕹️ Gameplay

- Control your spaceship and avoid incoming obstacles
- Survive as long as possible to achieve a high score
- Game speed increases over time for added difficulty

---

## ⚙️ Features

- 🚀 Smooth player movement and controls  
- 💥 Real-time collision detection  
- 🎯 Score tracking system  
- ⚡ Increasing difficulty over time  
- 🎮 Lightweight and runs directly in the browser  

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & layout  
- **JavaScript (Vanilla)** – Game logic & interactivity
